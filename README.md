## About The Project

This project exists to easily install libconf-2-4 for applications that require it.

Why use this method:
* Works on any Linux distribution.
* Saves time by installing from the latest repo.
* Removes the headache of searching for repos and files :smile:.

## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

### Installation

Installation is as easy as 1,2,3. Just follow the steps below.

1) Clone the repo
   git clone https://github.com/deadpicsel/libgconf-2-4-Installer.git

2. Mark the Script as executable
   chmod +x install.sh
3. Make a cup of Coffee or Tea and Enjoy!
